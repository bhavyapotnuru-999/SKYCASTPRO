# 🌦️ Weather Application

A modern and responsive Weather Application built using **HTML, CSS, and JavaScript**. This project allows users to search for any city and view real-time weather information through a clean and intuitive interface.

## 🚀 Features

* Search weather by city name
* Real-time weather data using a Weather API
* Display temperature, weather condition, humidity, and wind speed
* Dynamic weather icons
* Current date and time display
* Loading indicator while fetching data
* Error handling for invalid city names
* Responsive design for desktop and mobile devices
* Clean and modern user interface

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6+)
* Weather API

## 🔧 How It Works

1. Enter a city name in the search box.
2. Click the search button or press Enter.
3. The application sends a request to the Weather API.
4. Weather information is retrieved and displayed instantly.
5. Invalid city names trigger a user-friendly error message.
